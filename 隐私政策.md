# API调试助理隐私政策

### 版本更新日期：2025年12月10日

### 版本生效日期：2025年12月10日

API调试助理 是由 马强 （以下简称“我们”）为您提供，用于快速与主流的 API 调试程序建立连接，并协同进行 API 分析调试本机应用流量的应用。本隐私声明由我们为处理您的个人信息而制定。

我们不会解密、记录、存储、分享您的网络数据，而是将捕获到数据通过 Socks5 协议转发给主流的 API 调试程序程序。

我们不会记录、存储、分享您的任何个人信息，请放心使用。
